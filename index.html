Key Features

기능이 보이도록 만든 프로젝트 소개 페이지

실제 플레이 흐름과 시스템이 한눈에 들어오도록, 게임 UX 중심으로 내용을 정리했습니다.

01

직관적인 3x3 슬롯 보드

5개의 페이라인과 즉시 이해되는 인터랙션으로, 누구나 바로 시작할 수 있는 클래식 슬롯 플레이를 구현했습니다.

02

프리스핀과 꽝 게이지

스캐터 심볼과 연속 실패 누적 보상으로 플레이 템포를 조절해, 단순 반복 대신 기대감을 유지합니다.

03

하이스코어와 도감 수집

현재 보유 코인과 최고 기록을 비교하고, 등장 심볼을 도감으로 쌓아가며 장기 플레이 동기를 제공합니다.

04

매일 갱신되는 미션

스핀 횟수, 승리 횟수, 스캐터 수집, 연승 등 6개의 일일 미션으로 반복 플레이에 명확한 목표를 부여합니다.

Game Systems

배당과 진행 시스템을 함께 보여주는 구조

핵심 루프

  • 기본 코인 1,000에서 시작하고 베팅 단계를 조정합니다.
  • 당첨 시 즉시 코인을 누적하고 하이스코어를 갱신합니다.
  • 스캐터 3개 이상이면 프리스핀이 지급됩니다.
  • 미당첨이 누적되면 추가 프리스핀 보상이 열립니다.
  • 모든 심볼을 수집하면 도감 완성 상태를 기록합니다.

Project Notes

공개 페이지에서 강조할 포인트만 정리
